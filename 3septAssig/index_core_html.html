Q4. create an ordered list in html ............

  1. <html>: The root element that wraps the entire HTML document.
  2. <head>: Contains meta-information about the document, such as the title, character encoding, and links to external resources like stylesheets and scripts.
  3. <title>: Sets the title of the web page, which appears in the browser's title bar or tab.
  4. <meta>: Provides metadata about the HTML document, including character encoding and viewport settings.
  5. <link>: Links external resources like stylesheets to the HTML document.
  6. <body>: Contains the main content of the web page, including text, images, and other elements visible to the user.
  7. <h1> to <h6>: Headings with decreasing levels of importance, where <h1> is the highest and <h6> is the lowest.
  8. <p>: Represents a paragraph of text.
  9. <a>: Defines a hyperlink, allowing you to create clickable links to other web pages or resources.
  10. <img>: Embeds images on the web page.

Q5.Description of list.........

Full-Stack Web Development Tech Stacks

MEAN Stack
The MEAN Stack is a JavaScript-based tech stack consisting of MongoDB (NoSQL database), Express.js (web application framework), Angular (front-end framework), and Node.js (runtime environment). It's used for building dynamic and scalable web applications.
MERN Stack
The MERN Stack is another JavaScript-based tech stack that includes MongoDB (NoSQL database), Express.js (web application framework), React (JavaScript library for building user interfaces), and Node.js (runtime environment). It's used for building modern web applications with a focus on the user interface.
LAMP Stack
The LAMP Stack is a popular open-source tech stack comprising Linux (operating system), Apache (web server), MySQL (relational database management system), and PHP (server-side scripting language). It's commonly used for developing web applications, particularly in the PHP programming language.
